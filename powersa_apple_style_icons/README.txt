Powersa / Güçsa sidebar ikon seti

Format önerisi:
- Web / React / Vue / panel arayüzü: SVG kullanın. Ölçeklenir, retina ekranda net kalır.
- React Native / mobil / hızlı entegrasyon: PNG-1024 veya PNG-512 kullanın.

Dosyalar:
- svg/: Vektörel ikonlar, 1024x1024 viewBox
- png-1024/: Transparan köşeli 1024px PNG
- png-512/: Transparan köşeli 512px PNG
